# Sive Jozi — Personal Profile

Personal profile website built with React and Vite.

## Tech Stack

- React 18
- Vite
- CSS-in-JS (inline styles)

## Getting Started

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Deployment

Hosted on GitHub Pages. To deploy:

```bash
npm run build
npm run deploy
```

## Structure

```
src/
  Profile.jsx       # Main profile component
  App.jsx           # App entry point
  assets/           # Images
```
